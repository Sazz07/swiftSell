## Run Locally

Clone the project

```bash
  git clone https://github.com/Sazz07/swiftSell.git
```

Go to the project directory

```bash
  cd swiftSell
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run dev
```
